Pinterest publishing integration

Zenora uses a private internal publishing tool to create, schedule, and publish Pinterest Pins for Zenora-owned wellness music content.

The integration is used only by Zenora workflow operators. It is not a public app, a third-party service, an advertising product, or a user-data collection tool.

How the integration is used

  • Read Zenora Pinterest boards to choose appropriate publishing destinations.
  • Create Pins using Zenora-owned images, titles, descriptions, alt text, and YouTube links.
  • Maintain internal posting schedules and operational publishing logs.
  • Publish only Zenora-owned content prepared through Zenora's internal workflow.
